## Purpose
Warms-Paws helps pet owners take care of their pets during the winter. It provides essential services, tips, and products to keep pets safe, warm, and healthy.

## Key Features
- Browse local pet care services  
- Explore winter clothing and accessories for pets  
- Grooming options for different pets  
- Expert tips and advice for pet winter care  
- User authentication (login/register)  
- Dynamic service pages with detailed information  
- Smooth animations using AOS and interactive UI with Tailwind CSS  

## Technology Stack
- **Frontend:** React, Tailwind CSS, DaisyUI  
- **Backend / Authentication:** Firebase Auth  
- **Routing:** React Router DOM  
- **Notifications:** React Hot Toast  
- **Animations:** AOS (Animate on Scroll)  
- **Icons:** React Icons
